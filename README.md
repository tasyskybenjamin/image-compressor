# 图片压缩工具

这是一个简单高效的在线图片压缩工具，使用 Material Design 设计原则构建。

## 功能

- 支持拖拽上传和多文件选择
- 最多可同时处理10张图片
- 可调节压缩质量
- 实时预览压缩效果
- 支持批量下载压缩后的图片

## 技术栈

- HTML5
- CSS3
- JavaScript
- Material Design Web Components

## 如何使用

1. 克隆仓库到本地
2. 打开 `index.html` 文件在浏览器中查看
3. 拖拽图片或点击"选择图片"按钮上传图片
4. 调整压缩质量
5. 点击"下载压缩后的图片"按钮下载处理后的图片
